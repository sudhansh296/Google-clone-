# Google Homepage Clone

A clone of Google's homepage built with pure HTML and CSS.

## Features

- Google search bar with microphone and lens buttons
- AI Mode button
- App shortcuts (YouTube, ChatGPT, Hotstar, etc.)
- "Continue with these tabs" section
- Dark theme UI

## Tech Stack

- HTML
- CSS (Flexbox)

## How to Run

Just open `google-clone/index.html` in your browser.
